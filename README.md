# Gestión de Tareas – Backend (Spring Boot)

## Estructura del Proyecto

- **controller/**
  - Endpoints REST del sistema

- **service/**
  - Lógica de negocio (interfaces)

- **service/impl/**
  - Implementaciones de la lógica de negocio

- **repository/**
  - Acceso a datos mediante Spring Data JPA

- **entity/**
  - Entidades JPA y enumeraciones (enums)

- **dto/**
  - Objetos de transferencia de datos (DTO)

- **config/**
  - Configuraciones generales del proyecto (CORS, OpenAPI, etc.)

- **docs/uml/**
  - Diagramas UML del backend

- **docs/db/**
  - Script SQL para la creación de la base de datos
